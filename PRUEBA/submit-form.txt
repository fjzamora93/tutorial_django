Para enviar formularios, utilizaremos el siguiente action:

    <form action="{% url 'primer_proyecto:añadir' %}" method="post">

Como podemos ver, estamos usando una url relativa, que podremos sacar del nombre
del proyecto asignado en urls.py, y del propio nombre de la página.

En cuanto al método, ya sabemos que existen dos métodos: get y post.

PARA REALIZAR CAMBIOS
Si vamos a cambiar algo de la propia página, como añadir cosas, utilizamos POST.

PARA OBTENER DATOS
Si el objetivo es obtener datos de la página, una solicitud, por ejemplo, usamos GET.


SEGURIDAD Y CSRF
Para hacer un post, es necesario proporcionar las medidas de seguridad apropiadas.

Para eso nos iremos a SETTINGS, y ahí veremos un montón de MIDDLEWARE,
que se encargan de comprobar la seguridad. Así que para mantener la seguridad, será necesario
añadir un token.

Así que justo debajo de la primera etiqueta del form, añadimos esto:

    <form action="{% url 'primer_proyecto:añadir' %}" method="post">
        {% csrf_token %}


Pero esta no es la única forma de crear formularios... también podemos
crear formularios desde django. 
Para ello importaremos esto a nuestro archivo de views.py:

    from django import forms
